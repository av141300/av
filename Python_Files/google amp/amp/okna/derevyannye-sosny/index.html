Евроокна

Деревянные окна из сосны

Окна из сосны – это выбор для тех, кто ценит качество, красоту и долговечность. Рама производится по немецкой технологии из трехслойного клееного бруса, что обеспечивает прочность конструкции. Окна хорошо переносят перепады температур, не трескаются, не меняют форму. Общий срок эксплуатации достигает 80 лет.
